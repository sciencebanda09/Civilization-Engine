# Changelog

## 1.0.0 (2026-07-07)

- Initial release
- Multi-agent AI civilization simulation
- 6 scenarios (peaceful_valley, volcanic_winter, rich_valley, desert_oasis, island_colony, iron_fist)
- Interactive CLI game with council decisions, crisis responses, and tech direction
- 27 technologies across 4 eras with prerequisites tree
- Faction politics with rivalries and shifting influence
- Procedural terrain map with rivers and features
- Weather system with 4-season cycles
- Enemy tribes with raids, combat log, and taunts
- Achievement system (10 unlockable)
- Legacy system recording heroes across runs
- Multi-run analysis tool
- Groq API with 4-key load balancing + fallback provider
