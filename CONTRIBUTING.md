# Contributing

Thanks for your interest in Civilization Engine.

## Issues

- **Bug reports** — include the scenario, epoch count, and any error output
- **Feature requests** — describe what you want and why it fits a CLI civ sim

## Pull Requests

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Run `npm run lint` (zero errors required)
5. Submit a PR with a clear description of what changed and why

## Style

- No comments in code — let the code speak
- Match the existing patterns (see any file in `src/`)
- TypeScript, strict mode, ES modules
- No new dependencies unless absolutely necessary

## Running

```bash
npm install
npm run play
npm run lint
```
